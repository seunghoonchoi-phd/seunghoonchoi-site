# 리드패스트 (ReadFast) — 읽기 속도·이해 훈련

영어·중국어를 더 빨리, 정확하게 읽기 위한 설치형 웹앱(PWA).
상업 속독의 과장된 주장(시야폭 확장·RSVP 만능·1만 WPM·포토리딩)을 배제하고,
읽기과학·학습과학에서 **전이가 검증된** 기법만 구현했습니다.

## 핵심 원칙
- **병목은 눈이 아니라 어휘 자동화다.** 눈 운동은 읽기 시간의 ~10%. 지각폭(영어 식별 ~7–8자, 중국어 ~3자)은 고정이라 넓힐 수 없음. 빠른 독자는 시야가 넓어서가 아니라 단어를 빨리 알아봐서 빠르다.
- **속도는 이해로 게이트한다.** 헤드라인 지표는 ERR(유효 읽기속도) = 속도 × 이해율. 이해 60% 미만은 0점 → 훑기로 점수를 못 딴다.
- **커버리지 우선 → 속도 → 전략** 순서. 아는 단어 비율 95–98%를 먼저 채운다.
- **회귀·속발음은 보존**(억제하면 이해 손상). **개인 한계는 경험적으로 측정**(고정 WPM 목표 없음).
- 향상은 **학습하지 않은 새 지문(전이)**에서만 인정. 일반 인지능력 향상은 주장하지 않음(근거리 전이만).

## 레벨 시스템
첫 실행 때 언어별 레벨을 고른다(또는 3분 수준 측정으로 추천받는다). 레벨이 지문 난이도 창·훈련 시작 속도·어휘 빈도대·오늘의 코스 편성을 한 번에 정한다.

| 레벨 | EN | ZH | 특징 |
|---|---|---|---|
| 입문 | A2–B1 | HSK3–4 | 쉬운 지문·낮은 시작 속도, 어휘 커버리지 중심 |
| 중급 | B1–B2 | HSK4–5 | 일상 지문, ERR·반복읽기 중심 |
| 상급 | C1 | HSK5–6 | 고급 지문, 정복 모드·논문 3-패스 편성 |
| 최상급·과부하 | C2+ | HSK6+ | 최고난도 + 전공(재료과학) 지문, 한계 위 부하 → 회복으로 강화 |

레벨은 기본값일 뿐 잠금이 아니다 — 드릴 안에서 전체 난이도를 언제든 직접 고를 수 있고, 이해 90%+를 반복하면 앱이 레벨 올리기를 제안한다.

## 도장깨기 (정복 경로)
훈련 탭은 읽기과학의 스킬 사다리를 그대로 옮긴 **순차 정복 경로**다:

```
단어 인지(어휘) → [ZH: 글자 인지 → 단어 경계] → [EN: 청크 읽기] → 문장 검증
 → ERR 정독 → 반복읽기 전이 → 정복 모드 → 문맥 추론 → 모드 선택 → 미리보기
 → 자기설명·인출 → 최종 관문: 논문 3-패스        (EN 11단계 / ZH 12단계)
```

- 각 단계는 **측정된 기준 통과**로만 클리어된다(출석 아님): 예) 어휘 카드 25장 성숙, 청크 비계 1→2→3 점진 제거, ERR 이해 80% 3회+레벨 최상단 1회, 전이 유지, 정복 판정….
- 이전 단계를 깨야 다음이 열린다. **전 단계 정복 = 레벨 승급**, 최상급·과부하 도장까지 깨면 **완성**(이후 유지 모드 + 내 글 실전 전이).
- 승급하면 읽기 단계(정독·전이·정복)는 새 난이도 기준으로 다시 잠긴다 — 레벨마다 새 도장.
- 판정은 기존 훈련 기록에서 소급 파생(`js/progression.js`)이라 별도 저장이 없다. 설정에서 자유 훈련(잠금 해제)으로 전환 가능.

## 드릴 (13종, 트랙별)
- **커버리지** — 빈도순 어휘 간격반복(SM-2) + 어휘판단 반응시간 (정복 모드에서 표시한 미지 단어도 이 큐로 합류)
- **유창성 기초** — **청크 읽기**(phrase-cued, 비계 점진 제거·EN), **문장 검증 스피드**(SVT — 원문/변조 판정)
- **속도** — **정복(Conquer) 모드**(1차 독해→커버리지 게이트→반복 속독→전이), ERR 정독(이해도 게이트 + 적응형 페이서), 반복읽기 전이 검사
- **전략** — **문맥 추론**(클로즈 — 커버리지 갭 전략), 모드 선택(훑기/찾기/정독), 논문 3-패스(Keshav), 자기설명·인출(지연)
- **중국어 전용** — 단어 경계 분할, 글자 인지속도·의미 부수 (오답·느린 글자 재출제)
- **구조 활용** — 주제문 포레이징 (폭 확장 아님)
- **원리 탭** — 위 전부의 메커니즘·증거 강도·미신 반박·과부하→회복 프레이밍·출처

신설 3종(청크·문장 검증·문맥 추론)은 기존 지문을 재활용하고 정답이 원문으로 결정되는 과제 형식이라, 콘텐츠 추가 없이도 채점이 정직하다.

## 훈련 철학: 과부하 → 회복
한계 위 난이도로 과부하 → 회복으로 강화하는 방식을 따르되, **부하 축을 바로잡음**: 어휘·밀도·길이·인출에 과부하를 걸고 **눈 속도엔 걸지 않음**(이해 붕괴=훑기). 이 프리셋이 '최상급·과부하' 레벨이고, 다른 레벨은 같은 원리를 각자의 경계 바로 위에서 적용한다. 회복(간격·수면)은 새 단어 학습을 공고화하지 속도를 빠르게 하지 않음(정직 고지).

## 콘텐츠
모든 지문은 저작권 안전을 위해 **100% 원본 생성**, 이해 문항은 "지문만 보고 푸는" 검증을 거쳤습니다.
전공(재료과학) 지문은 domain 태그로 분리되어 최상급·과부하 레벨에서만 기본 노출됩니다.
사용자가 「내 글」에 붙여넣은 책·논문 단락의 이해 문항은 자동 클로즈(검증 안 됨, 자기 점검용)입니다.

## 실행 / 배포
- 로컬: `py _build/serve.py 5577` 후 http://127.0.0.1:5577 접속. (ES 모듈·fetch 때문에 file:// 직접 열기는 제한)
- **갱신(공개 후)**: `powershell -ExecutionPolicy Bypass -File "_build\sync.ps1"` — 앱 리포 푸시 + 홈페이지 `static/reading-trainer` 미러 + 사이트 카드까지 한 번에 동기화. 사이트 repo 경로는 `_build\site-path.local.txt`(UTF-8, gitignored)에서 읽음.
- 미러는 robocopy /MIR로 덮어쓰므로 **미러 쪽 파일을 직접 고치지 말 것** — 항상 앱 repo를 고치고 sync.
