When study material is scattered, searching starts to take more time than practice. The reason I made these apps is simple: gather what is needed in one place and make repetition easier.

This page gathers the learning apps and tools I built. The focus is work that needs repeated practice, such as Korean citizenship preparation, Korean typing, and English or Chinese reading training.

## Core Idea

A useful learning tool shows the next action clearly. Before reading a long explanation, the user should be able to solve a problem, type a sentence, or read a passage.

The citizenship app began with the problem of scattered material. Written, essay, and oral preparation is hard to repeat when everything lives in different places. The app puts the practice flow on one screen.

The typing trainer handles a body-skill problem. Even if the user has not memorized the Korean keyboard, the app should show the next key and finger clearly.

The reading trainer does not look only at speed. If comprehension falls, faster reading is not real reading skill. That is why the app checks what the user understood.

## Questions to Ask

- Is the user spending time finding material, or actually practicing.
- Is this material used once, or can it become a daily training tool.
- Does the app make the next action obvious.
- Is there a way to check speed, accuracy, or comprehension.
- Can the user open it on a phone and continue easily.

By this standard, the purpose of an app is not to add more features. It is to help the user repeat the right practice more often with less fatigue.
