사용자가 스타크래프트에서 숫자 키와 마우스를 빠르게 쓰면, 사용자는 다음 행동을 고르기 어려울 때가 있습니다. 이 문제는 손속도만으로 설명되지 않습니다. 사용자는 입력 순서를 고르는 동시에 게임 상황을 판단합니다. 두 선택이 겹치면 사용자가 다음 행동을 고르는 시간이 길어질 수 있습니다. 이 앱은 입력 속도만 점수로 계산하지 않습니다. 앱은 사용자가 입력하는 동안 유지한 판단 성적도 따로 측정합니다.

## 앱은 입력과 판단을 따로 측정한 뒤 함께 측정합니다

앱은 10분 세션에서 입력 기준선과 판단 기준선을 먼저 측정합니다. 사용자는 이후 입력과 판단을 함께 수행합니다. 앱은 세션 후반에 우선 과제와 판단 규칙을 바꾸고, 빨간 STOP 표적에서 사용자가 입력을 멈추는지도 확인합니다. 앱은 마지막 구간에서 입력 순서와 문제를 바꿔 사용자가 처음 보는 조건에 규칙을 적용하는지 측정합니다.

앱은 사용자가 숫자 키를 누른 뒤 표적을 클릭한 경우만 완료 동작으로 셉니다. 앱은 같은 키 반복과 잘못된 입력을 완료 동작으로 계산하지 않습니다. 사용자는 정확한 입력과 판단을 함께 유지해야 합니다. 높은 APM 수치만으로는 점수를 높일 수 없습니다.

## 앱은 결과를 세 값으로 나눠서 보여 줍니다

- 입력 속도 유지율은 앱이 동시 수행 속도를 입력 기준선과 비교한 값입니다.
- 판단 성적 유지율은 앱이 정확도와 반응 시간을 판단 기준선과 비교한 값입니다.
- 새 상황 판단 정확도는 앱이 처음 보는 판단 문제에서 계산한 정답률입니다.

앱은 입력 정확도와 판단 정확도가 기준을 넘을 때만 표적 간격을 줄입니다. 사용자가 훈련 화면을 떠나면 앱은 훈련 시간을 멈춥니다. 브라우저는 훈련 기록을 이 기기에만 저장합니다.

## 앱은 연구 결과를 반영했지만 게임 실력 향상을 보장하지 않습니다

이 앱은 두 과제를 함께 수행할 때 생기는 반응 지연과 반복 연습 결과를 다룬 연구를 참고했습니다. 앱은 사용자 수준에 맞춰 난이도를 조절하고, 연습과 다른 문제에서 다시 측정하는 방법도 훈련 순서에 반영했습니다.

연구자들은 이 앱이 스타크래프트 래더 티어를 올리는지 직접 시험하지 않았습니다. 사용자는 실제 게임 리플레이에서 교전 중 생산 공백 시간, 보급 한도 초과 시간, 주요 판단 누락 횟수가 줄었는지 따로 확인해야 합니다. 이 앱은 건강 상태, 지능, ADHD 증상을 검사하거나 치료하지 않습니다. 이 프로젝트는 Blizzard Entertainment와 관련이 없으며, Blizzard의 그림, 소리, 코드를 사용하지 않습니다.
