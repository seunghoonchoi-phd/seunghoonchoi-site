アメリカの理工系大学院に合格したからといって、それで終わりではありません。合格通知を受け取った瞬間から出発までの数か月は、思った以上にやることが多く締め切りも厳しいので、一つでも見落とすと入学・登録・ビザに支障が出かねません。この記事は University of Florida(UF) 大学院の入学・定着ガイドの第1回として、合格直後から出発までに必ず押さえておくべきことを順番にまとめます。フルファンディングのオファーレター(offer letter)を正確に読む方法、重要日程のカレンダー、プレイスメント試験と履修登録の手順、到着前の研究準備、そして出発前の韓国側の行政整理と送金までを一気に扱います。

## 全体像：合格から出発まで

入学が確定すると、「もう済んだこと」と「まだ残っていること」がはっきり分かれます。だいたい次の順番で進みます。

| 段階 | 主な項目 |
|---|---|
| 合格・書類 | オファーレターの受諾・署名返信 → I‑20 発給 → F‑1 ビザ面接・承認 |
| アカウント・行政 | GatorLink 作成 → ONE.UF Holds 解除(予防接種記録など) → 部屋の契約 |
| 出発前の仕上げ | 履修登録の相談 ・ 常備薬・診療書類 ・ 送金 ・ 国際運転免許(IDP) |
| 到着直後(0〜30日) | I‑94 ・ ISSS チェックイン ・ 銀行 ・ アメリカの電話番号 ・ SSN ・ 保険免除申請(waiver) |
| 定着後 | 免許の切り替え ・ クレジットカード ・ (任意) 車 ・ 学期ごと・毎年の繰り返し行政 |

合格してファンディングが確定し、ビザと I‑20 が出て、住まいとアカウントまで整えば、あとは三つの作業を順番に進めます。出発前には書類、薬、送金、国際運転免許を終わらせます。到着直後には ISSS チェックイン、銀行、SSN、免許、保険免除申請を処理します。学業ではプレイスメント試験と履修登録を確認します。今回の第1回は、このうち出発前に終わらせておくべきことに絞ります。

### 合格後すぐに片づける5つ

1. プレイスメント試験(Placement Exam)：入学前にオンラインのプレイスメント試験の受付が開きます(だいたい6月〜8月初め、eLearning)。75%以上を目標に受け、それ未満なら概論科目の履修が勧められます(卒業までの期間には影響なし、選択科目(elective)として認められます)。
2. オリエンテーション参加返信(RSVP)：学科・大学院オリエンテーションの参加返信の締め切りはだいたい8月初めです。日付・場所は GatorMail の通知で確認してください。
3. 指導教員が勧める事前学習：多くの教員が、到着前に関連論文を読み込み、研究ツールに習熟しておくことを勧めます。合流前に準備しておきましょう。
4. 履修登録：大学院の履修登録は One.UF だけで完結する方式ではありません。科目は指導教員とのミーティングで一緒に確定し、教員の署名をもらった登録フォームを学科に提出します(締め切り厳守、遅れると罰金)。
5. 継続中の治療薬・診療書類：服薬中であれば、英文の診療記録をあらかじめ発行してもらい、到着前に Student Health Care Center(SHCC、学生健康センター)に送っておくと、治療の空白を防げます。

## 1. ファンディングの実態：オファーレターの読み方

理工系の博士は、たいてい フルファンディング(GA, Graduate Assistantship: 助手契約)で合格します。「フルファンディング」が正確に何をカバーし、何が自分の負担なのかを知ることが、お金の管理の出発点です。以下の数字は *典型的な例* であり、オファーごとに異なるので、必ず自分のオファーレターで確認してください。

### もらうもの (Award)：典型的な GA パッケージの例

| 構成 | 例としての金額 | 備考 |
|---|---|---|
| Stipend(生活費) | 〜$32,000 / 年 | 隔週(biweekly)支給、指導教員の研究費/学科の財源 |
| 学費免除(Tuition Waiver) | 〜$22,000 | 年24単位(秋9 + 春9 + 夏6)基準、in/out‑of‑state の差額を含む |
| 健康保険(GatorGradCare) | GA 対象保険 | 学校が大半を補助しますが、本人負担の月額保険料と保険免除申請(waiver)の手続きは HR(Human Resources、人事部門)の案内を基準に確認 |

- 学費免除(tuition waiver)があっても、一部の手数料(fees)や保険関連の手続きは学生の責任として残ることがあるので、オファーレター・Bursar(学費・手数料の担当部門)・HR の案内をあわせて確認しましょう。
- 任用は 1年単位 で、毎年の研究・学業成果の評価で更新される仕組みが一般的です。

### 自分が払うもの (自己負担)

- 各種 fee 年およそ $1,900〜$2,900。学費とは別です。[UF Bursar](https://www.fa.ufl.edu/departments/bursar/)で確認・納付します。
- 手数料(fee)の締め切りを逃すと延滞料がつきます。GatorGradCare があっても Student Health Care Center の waiver を期限内に完了しなければならず、漏らすと UF の学生保険が学期(term)あたり $1,000+ で自動請求されることがあります。 これが事実上いちばん大きい「ミスの代償」です。

### 受諾の手順 (記録用)

`admissions.ufl.edu/appstatus` にログイン → VIEW UPDATE → オファー受諾 → 署名用紙を学科に返信 します。[CGS 4月15日決議](http://www.cgsnet.org/april-15-resolution)により4/15前の返信義務はありませんが、署名済みの用紙を返信して初めて受諾手続きが終わります。署名済み用紙が漏れると I‑20 の発給が止まることがあるので、最後まで確認してください。

## 2. 重要日程カレンダー (秋学期(Fall)の例)

すべての時刻は アメリカ東部時間(ET, Eastern Time) 基準です。韓国時間(KST, Korea Standard Time)はだいたい ET +13〜14時間です。以下は秋学期を基準にした例で、毎年の公式日程で再確認してください。

| 時点 | 日程 | メモ |
|---|---|---|
| 6月〜8月初め | プレイスメント試験 受付 | eLearning オンライン、75%以上が目標 |
| 8月初め | GradStart(大学院オリエンテーション) Canvas 学習モジュール開放 | [gradadvance](https://gradadvance.graduateschool.ufl.edu/programs/orientation/) |
| プログラム開始30日前〜 | アメリカ入国可能 | I‑20 *Earliest Admission Date*(最も早い入国可能日)以降のみ |
| 8月初め | 学科オリエンテーション参加返信(RSVP)締め切り | GatorMail の通知を確認 |
| 8月中旬 | 任用・プログラム開始日 | 初給与は1か月ほど後に入金 |
| (8月中) | ラボ合流日を指導教員と相談 | 定着(電話・車・SSN, Social Security Number)の進み具合を見ながら |
| 8月中旬 | 学科オリエンテーション ・ 朝食提供 | プログラム要件の説明 |
| 8月中旬 | 工学部 新入 PhD Welcome | [工学部の案内](https://www.eng.ufl.edu/graduate/herbert-wertheim-college-of-engineering-new-ph-d-student-welcome/) |
| 履修登録締め切り | だいたい開講初週 | 履修登録が遅れると罰金($100) |
| 開講直後 | Drop/Add(履修修正)期間 | 修正可能 |
| I-20 プログラム開始日から30日以内 | ISSS(International Student Services) オンラインチェックイン 3ステップ | 到着後の手続き |
| 翌年春 | 確定申告(Form 8843 など税金関連書類) | 毎年繰り返し |

流れで見ると次のようになります。

```
出発前(韓国)        到着直後(0〜30日)            定着後/学期中
─────────────────────────────────────────────────────────────
ビザの実物 · IDP(国際運転免許) → I-94 · ISSS チェックイン · 銀行 → 免許の切り替え · クレジットカード
英文の診療書類                  → アメリカの電話番号 · SSN 申請  → 保険免除申請(毎学期)
常備薬 · 送金                  → 保険登録の確認 · 予防接種 hold → 確定申告(毎年)
韓国側の行政整理     →  オリエンテーション · 履修登録      →   車(任意)
```

## 3. プレイスメント試験 (Placement Exam)

- 受付は6月〜8月初め、[eLearning](https://elearning.ufl.edu/) のオンラインで行われます。選択式+記述式の構成です。
- 範囲: 結晶学・欠陥・機械的物性・状態図・電気/磁気物性(材料概論の標準)。教材は Callister(*Materials Science and Engineering: An Introduction*、材料工学概論の教科書)などどの版でも構いません。
- 75%未満または未受験 → 概論科目(EMA 6001、材料工学概論)の履修が強く勧められます。EMA 6001 は選択科目(elective)として認められるので 卒業までの期間には影響ありません。材料工学の学部出身者にも受験が勧められます。

## 4. 履修登録の手順：One.UF で直接申請はしません

大学院の履修登録は One.UF だけで完結する方式ではありません。次のフォーム手続きで進めます。

1. 科目の確認: [one.uf.edu/soc](https://one.uf.edu/soc)で開講・時間を確認します(ログイン)。
2. 指導教員との初回ミーティングで科目を一緒に確定 → 教員がその場で登録フォームに署名 します。一人で科目を確定・提出するのではなく、指導教員または graduate coordinator(大学院の行政担当者)の承認を経る仕組みです。
3. 最新の Graduate Registration Form(大学院履修登録フォーム) を使います([MSE Forms](https://mse.ufl.edu/academics/forms/))。科目名・番号・単位を正確に記入します。
4. @ufl.edu のメールから 承認されたフォームを advisingforms@mse.ufl.edu に送ります(個人メールは不可)。
5. 締め切りを逃すと罰金($100)がつくことがあります。 入国からオリエンテーションまでの間に、教員ミーティング・署名・提出まで終えられるよう日程を組むほうが安全です。

## 5. 到着前の研究準備

- 到着前の課題: 多くの指導教員が、到着前に 関連論文の読み込み + 研究ツール・手法への習熟 を勧めます。合流前に把握して、初回ミーティングに持っていく内容を準備しておくとよいです。
- ラボ合流日は柔軟 です: 到着後、電話・車・SSN など定着の進み具合を見ながら 指導教員とのミーティングで実際の開始日を決めます(任用の公式日とは別になることがあります)。
- 使うツールは研究室ごとに違うので、自分の研究室で使うツールは入学後に確認すれば大丈夫です。UF は HiPerGator(高性能スーパーコンピュータ)などの計算リソースも提供しています。

## 6. 出発前の韓国側の行政整理

出発前に韓国で整理しておくとよい行政手続きです。

- 国民健康保険: 3か月以上の国外滞在の場合、出国の翌日から 給付停止(保険料免除) の届け出ができます。
- 国民年金: 国内の所得がなければ 納付例外 を申請できます。
- 携帯電話: 「番号維持(一時停止)」または格安SIMの最低料金プランを活用できます(銀行/カカオの OTP 用)。
- 郵便物: 郵便局の転送サービスを申し込むか、家族の住所で受け取れるよう整理しておきます。

## 7. 送金 (韓国 → アメリカ)

- 2026年基準の無証憑(むしょうひょう)海外送金の上限は、全金融機関合算で年 米ドル10万ドル です([企画財政部の発表](https://www.korea.kr/news/policyNewsView.do?newsId=148956081))。ただし銀行ごとの非対面の上限、国税庁への通報基準、留学生の証憑送金は別途確認が必要で、送金の直前に自分の銀行で再確認するほうが安全です。
- Wise(為替・手数料が有利、少額・定期) / 電信為替 SWIFT(大きな金額・学費) / 留学生の証憑送金(上限の枠外)を、状況に合わせて選べます。
- アメリカの口座を作る前は、Wise カードやトラベルカード で現地の ATM から引き出して、初期の生活費をまかなえます。

## 出発前(韓国)チェックリスト：第1段階

最後に、出発前に下の項目をもう一度確認してください。

- アカウント・学校の行政：GatorLink、ONE.UF holds(未解決の行政項目)、予防接種記録
- 住まい・現地とのつながり：部屋の契約、KSA(韓国人留学生会)とのつながり、空港での短期SIM
- 授業・研究：履修登録の相談
- 健康・書類・生活：精神科・慢性疾患の英文書類、到着直後までつなぐ薬、常備薬、海外配送、国際運転免許(IDP)の発給、ビザの実物確認

ここまでが、合格してから出発までの全体像です。次回は、到着直後の30日以内に終わらせるべき行政(I‑94、ISSS チェックイン、銀行、SSN、保険免除申請)を順番に扱います。
